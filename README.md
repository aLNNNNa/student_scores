# 学生成绩管理系统（基础版）

一个命令行下的学生成绩管理工具，支持添加、查询、统计、成绩分布图等功能。  
**当前版本**：数据仅存储在内存中（程序关闭后数据丢失），代码全部使用已学过的 Python 基础语法，无第三方库。

## ✨ 已实现功能

- 添加学生（姓名、数学、语文、英语成绩，0-100 验证）
- 查看所有学生（表格形式显示各科成绩与总分）
- 按姓名查询单个学生（显示详细成绩、总分、平均分）
- 统计信息：
  - 班级人数
  - 每科平均分、最高分、最低分
  - 总分前三名（按总分降序）
- 成绩分布柱状图（字符版，按平均分分段统计）
